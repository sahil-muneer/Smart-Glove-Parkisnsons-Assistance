# 🧤 Smart Glove for Parkinson's Patients

## 📌 Overview
This project is an assistive IoT device designed to help people with Parkinson’s disease by reducing hand tremors and improving daily activities like eating.

---

## ⚙️ Components Used
- ESP32
- MPU6050 (Accelerometer + Gyroscope)
- Servo Motors
- Power Supply

---

## 🔧 Working Principle
The MPU6050 sensor detects unwanted hand tremors in real-time.  
The ESP32 processes this data and sends corrective signals to servo motors, which stabilize the movement of the hand.

---

## 🚀 Features
- Real-time tremor detection  
- Motion stabilization  
- Low-cost assistive solution  
- Portable design
